# JavaScript Array Programs

This repository contains various array programs implemented using JavaScript. These programs helped me practice array traversal, searching, sorting, and problem-solving skills.

## Programs Covered

### 1. Find Largest Number in an Array
Finds the maximum element from an array.

### 2. Find Smallest Number in an Array
Finds the minimum element from an array.

### 3. Sum of Array Elements
Calculates the total sum of all elements in an array.

### 4. Average of Array Elements
Calculates the average value of all elements in an array.

### 5. Reverse an Array
Reverses the elements of an array.

### 6. Sort an Array
Sorts array elements in ascending or descending order.

### 7. Remove Duplicate Elements
Removes duplicate values and returns unique elements.

### 8. Find Second Largest Number
Finds the second highest number in an array.

### 9. Find Second Smallest Number
Finds the second lowest number in an array.

### 10. Left Rotate Array by K Positions
Rotates array elements to the left by K positions.

### 11. Right Rotate Array by K Positions
Rotates array elements to the right by K positions.


## Technologies Used

* JavaScript (ES6)
* HTML
