## Programs Covered

### 1. Find Largest Number in an Array

### 2. Find Smallest Number in an Array

### 3. Sum of Array Elements
Calculates the total sum of all elements in an array.

### 4. Average of Array Elements
Calculates the average value of all elements in an array.

### 5. Reverse an Array
Reverses the elements of an array.

### 6. Sort an Array

### 7. Remove Duplicate Elements

### 9. Find Second Smallest Number

### 11. Right Rotate Array by K Positions

### 12. Flatten Array
Flatten array is concept in javascript that flat the multi-nested array into the single dimensional array.

for example:--

let arr = [1,2,[3,4],[5,6]];

let ans = arr.flat();

console.log(ans);

## output:1,2,3,4,5,6

Why we use flatten array?

Sometimes data come from the api or complex structure in nested form.

let student = {
    ["A","B"],
    ["C","D"],
    ["E","F"]
};

## To work with this all student easilt we flat the array.

let flattenans = student.flat();

console.log(flattenans);
